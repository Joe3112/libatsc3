AIRWAVZ-README 3/5/2020

To build this project please place a copy of the Airwavz RedZone Receive Android SDK in the following directory:
sample_app_no_phy/Airwavz-RZR-SDK including at least the include and bin folders.

Place another copy of the contents of the bin directory from the SDK into
    sample_app_no_phy/app/src/main/jniLibs/
Copy both the armeabi-v7a folder and the arm64-v8a folder if you want support for both 32 and 64 bit.
